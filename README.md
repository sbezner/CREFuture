# CREFuture Website

A visionary digital platform exploring the transformative collision of Artificial Intelligence and Commercial Real Estate.

## About

CREFuture champions **Tech Optimism**—the belief that AI empowers professionals rather than replacing them. We explore how the Augmented Professional is reshaping the industry.

## Three Core Pillars

- **The Future of the Workforce** - How specific real estate roles are evolving with AI
- **Functional Transformation** - How AI is rewriting industry processes and operations
- **The Global Tech Stack** - Curated technologies moving from hype to industry standards

## Tech Stack

- **Framework**: [Astro](https://astro.build)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Hosting**: GitHub Pages (static site)

## Development

### Prerequisites
- Node.js 16+ and npm

### Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Project Structure

```
src/
├── layouts/
│   └── Layout.astro       # Main layout component
├── pages/
│   ├── index.astro        # Home/landing page
│   ├── workforce.astro    # Workforce pillar
│   ├── transformation.astro # Transformation pillar
│   ├── tech-stack.astro   # Tech stack pillar
│   └── about.astro        # About page
└── styles/
    └── global.css         # Global styles and Tailwind imports
```

## Deployment

The site is deployed to GitHub Pages via GitHub Actions. Push to `main` or `claude/cre-future-website-ML6Pb` branch to trigger automatic builds and deployments.

**Live URL**: https://sbezner.github.io/CREFuture/

## Content

Content pages are structured with placeholder sections ready for detailed research and writing:

- **Workforce**: Evolving roles, augmented professional mindset, reskilling opportunities
- **Transformation**: Autonomous operations, predictive leasing, real-time valuations
- **Tech Stack**: Core AI technologies, enterprise agents, data infrastructure

## Design

The site features a bold, futuristic aesthetic with:
- Vibrant color palette (primary cyan, secondary magenta, accent yellow)
- Gradient text effects
- Interactive hover states
- Responsive design for all devices
- Smooth scrolling and animations
